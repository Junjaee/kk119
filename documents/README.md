## 🚀 프로젝트 정보

**프로젝트명**: 교권119  
**목표**: 교사들의 교권 보호를 위한 신고 및 법률 상담 시스템  
**일정**: 2025년 8월 28일 ~ 9월 30일  
**시연**: 2025년 9월 30일 토론회  

## 💻 기술 스택

- **Frontend**: Next.js 14, React 18, Tailwind CSS
- **Backend**: Next.js API Routes (서버리스)
- **Database**: Supabase (PostgreSQL)
- **Auth**: Supabase Auth
- **Storage**: Supabase Storage
- **Hosting**: Vercel
- **Mobile**: PWA

## 🎯 핵심 기능

1. **교권 침해 신고 시스템**
   - 익명 신고 접수
   - 증거자료 첨부
   - 진행 상태 추적

2. **변호사 법률 상담**
   - 전문 변호사 매칭
   - 온라인 법률 자문
   - 사건 관리

3. **교사 커뮤니티**
   - 완전 익명 보장
   - 사례 공유
   - 상호 지원

4. **관리자 대시보드**
   - 통계 모니터링
   - 변호사 관리
   - 시스템 운영

## 👥 사용자 유형

- **교사**: 신고 작성, 커뮤니티 참여
- **변호사**: 법률 상담, 사건 관리
- **관리자**: 시스템 운영, 통계 관리
- **비회원**: 제한적 콘텐츠 열람
